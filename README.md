# web-scraping-challenge
 
Web scraping application designed to collect data about Mars and assemble it on a web page using Flask to render an html template. Original data collected in Jupyter Notebook environment, then sent to a .py script and finally imported as a dictionary to index.html and rendered on localhost via Flask.
